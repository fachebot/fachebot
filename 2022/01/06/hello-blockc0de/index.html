Hello Blockc0de

发表于 2022-01-06 更新于 2022-01-12 分类于 blockc0de

项目介绍

Blockc0de 是一款支持 0 代码开发区块链应用的工具，用户通过链接图形节点来处理业务逻辑的所有需求，而不需要任何编码技能。一旦用户准备好部署Graph，可以将其直接上传至网络并运行，借助 Blockc0de 用户可以建立可视化、自动化任务。

任何人都可以在 Blockc0de 中创建Graph，Graph由多个不同类型的节点和链接组成。每个节点至少有一个接受要处理的值的输入和一个产生结果的输出。将一个输出连接到一个输入建立一个数据路径，使一个节点能够将数据输入到另一个节点。用这个简单而有力的原则可以实现任何复杂的业务逻辑。例如：监听链上鲸鱼转账并通过Telegram或者Twitter发送通知。

Hello World

Hello World是一个最著名的程序。对每一位程序员来说，这个程序几乎是每一门编程语言中的第一个示例程序。下面我也试着用 Blockc0de 实现一个Hello World的Graph。

  1. 首先打开 Blockc0de 在线 IDE，点击右上角 Connect to wallet 使用 MetaMask 钱包进行登录。

  2. 点击左侧节点列表里面的 Entry Pont 节点，会自动在Graph上面创建一个 Entry Pont 节点，Entry Pont 是Graph执行的入口，类型其它编程语言中的 main() 函数。

  3. 创建一个 Print 节点，并将 Entry Pont 节点的 Out 与 Print 节点的 In 进行链接，表示 Entry Pont 执行完成之后应立即执行 Print 节点。Print 节点可以将 Message 参数打印输出。

  4. 我们可以看到 Print 节点还需要接收一个名为 Message 的参数，现在我们再创建一个 String 节点，它可以存储一个字符串类型的值。创建完成后我们输入文本 Hello World，并将它与 Print 节点的 Message 参数进行链接。

  5. 现在我们的第一个Graph已经编写完成了，点击顶部菜单栏 Execution/Launch 按钮即可上传至服务器并执行。

  6. 最后在右下角 Terminal 处输出 Hello World，大功告成。

当然 Blockc0de 还能在不编写任何代码的情况下实现更加复杂的业务逻辑，我们留着以后慢慢探索吧。
